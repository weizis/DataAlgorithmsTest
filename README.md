# DataAlgorithmsTests

This project contains **MSTest unit tests** for the `DataAlgorithms` library.

Test coverage includes all implemented sorting and searching algorithms.


## Tested Features

**Sorting**
- BubbleSort
- QuickSort
- MergeSort

**Searching**
- LinearSearch
- BinarySearch

## Notes

- Each sorting/search algorithm has multiple test cases to verify correctness.  
- This project references the main `DataAlgorithms` library.  
- Add more tests if new comparers or algorithms are implemented.

## Running Tests

Make sure you have the **.NET 9 SDK** installed.  

Run all tests from the terminal inside the `DataAlgorithmsTests` folder:
```bash
dotnet test DataAlgorithmsTests.csproj
```
